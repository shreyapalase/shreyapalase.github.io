# Contributing Guidelines

## Important Notice

This repository is a **personal portfolio project** and is maintained solely by the owner.

## No Contributions Allowed

At this time, we do **not accept contributions, pull requests, or external modifications**.

This includes but is not limited to:
- Bug fixes
- Feature additions
- Design changes
- Documentation updates

Any pull request submitted will be closed without review.

## Forking Policy

While this repository may be publicly visible for demonstration purposes, **forking is not permitted for reuse, redistribution, or modification of any kind**.

## Usage Restrictions

The code, design, and assets in this repository are strictly for viewing purposes only.
You may not copy, reuse, or repurpose any part of this project.

## Contact

If you wish to discuss the project or request permission for any use, please contact the repository owner directly.
